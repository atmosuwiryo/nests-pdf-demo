## Description

A NestJS application that demonstrates PDF generation for business quotations using the `@toondaey/nestjs-pdf` library with Handlebars templating.

## Features

- PDF quotation generation with customizable templates
- REST API endpoints for PDF generation and sample data
- Input validation using class-validator
- Swagger API documentation
- Static file serving for images referenced in PDFs

## Project setup

```bash
$ pnpm install
```

## Configuration

Create a `.env` file in the root directory with the following variables:

```
NODE_ENV=Dev
VERSION=0.0.1
PORT=3000
HOST=http://localhost:3000
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## API Documentation

Once the application is running, you can access the Swagger UI documentation at:

```
http://localhost:3000/docs
```

## PDF Generation

The application provides the following endpoints for PDF generation:

- `POST /quotation/generate-pdf` - Generate a PDF from custom data
- `GET /quotation/default-data` - Retrieve sample quotation data

## Static Files

Place image files in the `public/images` directory. When referencing company logos in the quotation data, only provide the filename as the application will automatically prepend the host URL and path.

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
